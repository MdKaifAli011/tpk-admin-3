Motion of Blocks Connected by String – Complete Notes, Revision, Important Questions & Downloads

Analyses two or three blocks on a horizontal surface connected by a massless string, driven by an external force F. Unlike blocks in contact (where the connecting force is a contact/compression force), here the connection is a tension T in the string. Common acceleration a = F/(m₁+m₂), tension T = m₁ F/(m₁+m₂) for the block farthest from F. For three-block systems, two independent tension values T₁ and T₂ can be derived for each string segment.

Preparation Strategy

1

Whole-System Acceleration First Compute a = F/(m₁+m₂+...+mn) using the full system treated as a single mass. String tensions are internal and cancel at this stage.

2

Isolate the Sub-Block for T For the string tension between blocks A and B, draw the FBD of the block farthest from F. Net force on it = T = mᵣ̵ × a, where mᵣ̵ = mass(es) on the far side of the string.

3

Distinguish from Contact Force Strings can only pull (tension). Contact forces can only push (compression). In string problems, the connecting force is tension T; in contact problems, it is a normal/contact force f. The formulas are structurally identical but the physical mechanism differs.

Quick Revision Cards

Concept → Trap → Example

1) Two Blocks Connected by String

Core Formula

F applied on m₂, with m₁ connected by string. System acceleration a = F/(m₁+m₂). String tension (which must accelerate only m₁ on the far side): T = m₁ F / (m₁+m₂).

  • a = F/(m₁+m₂)
  • T = m₁ F / (m₁+m₂)
  • T is less than F; the string only pulls the front block
Example (NEET-style)m₁ = 2 kg, m₂ = 3 kg, F = 25 N: a = 5 m/s²; T = 2×5 = 10 N.

2) Three Blocks — T₁ and T₂

NEET Exam Target

Three blocks m₁ (front), m₂ (middle), m₃ (rear) with strings T₁ (between 1–2) and T₂ (between 2–3). F on m₃ (rear end). Common a = F/(m₁+m₂+m₃). T₂ = (m₁+m₂) F / total mass. T₁ = m₁ F / total.

  • T₂ = (m₁+m₂) F / (m₁+m₂+m₃)
  • T₁ = m₁ F / (m₁+m₂+m₃)
  • T₂ > T₁ always (since T₂ must pull more mass)
Example (NEET-style)m₁ = 1, m₂ = 2, m₃ = 3 kg, F = 12 N: a = 2 m/s²; T₁ = 2 N; T₂ = 6 N.

3) Force Applied on Front Block (Reversed)

Direction Switch

If F acts on the front block m₁ (instead of the rear), the acceleration remains the same but the string tension formula now contains the rear block mass(es).

  • a = F/(m₁+m₂+m₃) — unchanged
  • T₁ (string between 1 and 2) = (m₂+m₃) F / (m₁+m₂+m₃)
  • T₂ (string between 2 and 3) = m₃ F / (m₁+m₂+m₃)
Example (NEET-style)F = 12 N on m₁ = 1 kg, m₂ = 2 kg, m₃ = 3 kg: T₁ = 10 N; T₂ = 6 N.

US Curriculum Bridging

AP Physics C covers two-body tension well. NEET extends to three strings and tests whether a student can correctly identify which sub-system to isolate.

Three-String Tension Identification

US physics problems rarely present three blocks and ask for a specific one of two tensions. NEET does this regularly, requiring systematic isolation of the correct sub-group.

  • Draw the full system diagram first; label every string with its own tension symbol.
  • For each string, isolate the sub-group on one side and write Newton’s 2nd Law for only that sub-group.

Tension vs Contact Force Conceptual Clarity

Strings can only be under tension (pull). A rope that would go into compression simply goes slack. Contact forces (adjacent blocks touching) can only push. NEET tests this conceptual distinction explicitly.

  • If the string goes slack (T = 0), the blocks move independently.
  • Contact force = 0 means blocks separate. Both limiting cases appear in NEET MCQs.

Frequently Asked Questions

Notes · Downloads · Revision · Important Questions
Why does the tension formula look the same as the contact force formula?
Both result from Newton’s 2nd Law applied to a sub-system: force = sub-system mass × common acceleration. The structural form is identical. The physical difference is that contact forces are compressive (push), while string tensions are tensile (pull). This matters when the problem involves ropes that could go slack.
Can I have different tensions in the same string at different points?
For a massless string (as assumed here), no. The tension is uniform throughout any section of a massless string. For a massive string, tension varies with position. That case is covered separately in the next topic (Motion of Massive String).
What happens if a string breaks?
If the string carrying T₂ breaks, the sub-group of blocks it was pulling separates and continues with the acceleration it had at that instant. The remaining blocks re-equilibrate at a new (higher) acceleration since less total mass is being driven.
