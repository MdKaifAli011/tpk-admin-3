Minimum Mass Hung from String – Complete Notes, Revision, Important Questions & Downloads

A mass m₂ hangs over a frictionless pulley and pulls a mass m₁ resting on a rough horizontal or inclined surface. The minimum m₂ needed to just start the motion of m₁ is found at the limiting friction condition. For horizontal surface: m₂ = μm₁. For inclined surface (angle θ): m₂ = m₁(sinθ + μ cosθ). NEET tests both variants with numerical substitution and conceptual MCQs.

NEET Mastery Strategy

1

Draw FBD for Each Mass Separately Mass m₂ (hanging): T = m₂g at limiting equilibrium. Mass m₁ (on surface): T = friction force at limiting condition. For horizontal: T = μm₁g ⇒ m₂g = μm₁g. For inclined (m₁ pulled up slope): T = m₁g(sinθ + μ cosθ). Always equalise the tension T.

2

Identify Friction Direction Friction opposes the tendency of motion of m₁. When m₂ pulls m₁ via string: friction on m₁ acts away from m₂ direction. On incline where m₁ is pulled up: friction acts down-slope (opposing up-slope tendency), so both gravity (sinθ) and friction (μ cosθ) resist the pull. Hence m₂ = m₁(sinθ + μ cosθ).

3

Check If Problem Asks for Minimum or Maximum m₂ Minimum m₂ to START motion = limiting friction condition. If the question asks for maximum m₂ that still keeps m₁ stationary (e.g. m₁ on incline tending to slide DOWN), friction acts up-slope and resists gravity down-slope, so T + Fl = m₁g sinθ and m₂ = m₁(sinθ − μ cosθ). The sign depends on problem geometry.

1) Horizontal Surface Case

Must Memorise

Setup: mass m₁ on rough horizontal surface connected via string over frictionless pulley to hanging mass m₂. Limiting friction on m₁ = μm₁g. At the moment of starting motion, tension T = m₂g equals limiting friction: m₂g = μm₁g. Therefore minimum m₂ = μm₁.

  • Minimum m₂ = μm₁ (horizontal rough surface)
  • Condition: T = m₂g = μm₁g at start of motion
  • Pulley is frictionless: tension same throughout string
  • Normal reaction N = m₁g (horizontal surface, no vertical pull)
  • Limiting friction Fl = μN = μm₁g
Example (NEET-style)m₁ = 3 kg, μ = 0.4. Minimum m₂ = 0.4 × 3 = 1.2 kg. If m₂ = 1.2 kg is hung, the system is just about to move. Any m₂ > 1.2 kg starts motion.

2) Inclined Surface Case

Must Memorise

Mass m₁ on rough inclined plane (angle θ) connected via string over frictionless pulley at top to hanging mass m₂. String pulls m₁ up the slope. At limiting condition: T = m₁g sinθ + μm₁g cosθ (both gravity component and friction oppose upward motion of m₁). With T = m₂g: minimum m₂ = m₁(sinθ + μ cosθ).

  • Minimum m₂ = m₁(sinθ + μ cosθ) (inclined plane, m₁ pulled up slope)
  • Gravity component down slope: m₁g sinθ
  • Limiting friction (down slope, opposing upward tendency): μm₁g cosθ
  • Normal reaction N = m₁g cosθ (perpendicular to incline)
  • At θ = 0° (horizontal): formula reduces to m₂ = μm₁ ✓
Example (NEET-style)m₁ = 5 kg, θ = 30°, μ = 0.3. sin 30° = 0.5, cos 30° = 0.866. m₂ = 5(0.5 + 0.3×0.866) = 5(0.5 + 0.26) = 5×0.76 = 3.8 kg. The system just starts motion when m₂ = 3.8 kg.

3) Limiting Condition and Coefficient of Friction

Key Relation

From the horizontal case at limiting condition: μ = m₂/m₁. This provides a method to measure the coefficient of friction experimentally by determining the minimum hanging mass m₂ that just starts m₁. The ratio m₂/m₁ at just-start-of-motion = μ. This is a commonly tested conceptual point in NEET MCQs.

  • μ = m₂/m₁ (from horizontal case at limiting condition)
  • Experimental method: find smallest m₂ that starts motion; μ = m₂/m₁
  • On incline: μ = (m₂/m₁ − sinθ)/cosθ
  • Always use limiting (static) friction for ‘just starts motion’ condition
  • Use kinetic friction only if asked about motion after it has started
Example (NEET-style)Experiment: m₁ = 2 kg on horizontal rough surface. Minimum hanging mass to start motion = 0.6 kg. μ = 0.6/2 = 0.3. The surface has coefficient of static friction = 0.3.

US Curriculum Gaps

AP Physics and US curricula vs NEET scope on pulley-friction two-mass problems.

AP Physics Covers Atwood but Not Friction-Pulley Combined

AP Physics 1 covers Atwood machines and inclined planes with friction separately, but less emphasis on the combined ‘minimum mass to start motion’ setup. NEET tests direct numerical application: given m₁, μ, and θ, calculate minimum m₂. Practise the formula-application sequence: (1) FBD → (2) T = m₂g → (3) T = Fl at limiting.

  • Horizontal: m₂ = μm₁ (very quick once limiting condition is set).
  • Inclined: m₂ = m₁(sinθ + μ cosθ) — both weight component and friction.

Sign Convention for Inclined Case

The sign in the inclined formula (sinθ ± μ cosθ) depends on the geometry. If m₂ pulls m₁ UP the incline (typical NEET setup): m₂ = m₁(sinθ + μ cosθ). If m₁ slides DOWN the incline and m₂ opposes: m₂ = m₁(sinθ − μ cosθ). Read each problem carefully for the direction of impending motion of m₁.

  • m₁ pulled up slope: both mg sinθ and friction μmg cosθ oppose → + sign.
  • m₁ tends to slide down: friction is up-slope, assists the string → − sign.

Frequently Asked Questions

Notes · Downloads · Revision · Important Questions
Why do we set T = m₂g for the hanging mass at limiting condition?
At the instant of starting motion (limiting equilibrium), the acceleration is zero. For the hanging mass m₂: net force = m₂g − T = 0 ⇒ T = m₂g. This is equal to the weight of m₂. As the system begins to move, acceleration becomes non-zero, but for the ‘just starting’ (minimum mass) condition, the system is in limiting equilibrium with zero acceleration.
Is the normal reaction on m₁ equal to m₁g in both horizontal and inclined cases?
For horizontal surface: N = m₁g (weight balanced directly by normal reaction). For inclined surface: N = m₁g cosθ (only the perpendicular component of weight is balanced by normal reaction). This is why the friction on inclined surface is μm₁g cosθ, not μm₁g.
What happens if m₂ is larger than the minimum value?
If m₂ exceeds the minimum value, the net force on the system becomes non-zero and the system accelerates. In this case, kinetic friction acts on m₁ (not static limiting friction). The acceleration is calculated using T − μm₁g = (m₁+m₂)a — this is a different calculation from the minimum condition. NEET usually tests specifically the ‘just starts’ condition using limiting (static) friction.
If the pulley is not frictionless, how does the calculation change?
A frictionless pulley ensures the tension T is the same throughout the string (both sides equal T = m₂g). If the pulley has friction, the tensions on the two sides differ: T₂ on the hanging side and T₁ on the m₁ side, with T₁ < T₂ or T₁ > T₂ depending on the rotation direction. NEET problems almost always specify frictionless pulley, so T is equal on both sides.
