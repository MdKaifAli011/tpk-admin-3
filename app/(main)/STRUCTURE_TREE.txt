═══════════════════════════════════════════════════════════════
                    ROUTE STRUCTURE
═══════════════════════════════════════════════════════════════

Static Routes:
  /                    → page.js (Homepage)
  /login               → login/page.js
  /register            → register/page.js
  /contact             → contact/page.js

Dynamic Routes (Hierarchical):
  /[exam]                                    → [exam]/page.js
  /[exam]/[subject]                          → [exam]/[subject]/page.js
  /[exam]/[subject]/[unit]                   → [exam]/[subject]/[unit]/page.js
  /[exam]/[subject]/[unit]/[chapter]         → [exam]/[subject]/[unit]/[chapter]/page.js
  /[exam]/[subject]/[unit]/[chapter]/[topic] → [exam]/[subject]/[unit]/[chapter]/[topic]/page.js
  /[exam]/[subject]/[unit]/[chapter]/[topic]/[subtopic]
    → [exam]/[subject]/[unit]/[chapter]/[topic]/[subtopic]/page.js
  /[exam]/[subject]/[unit]/[chapter]/[topic]/[subtopic]/[definition]
    → [exam]/[subject]/[unit]/[chapter]/[topic]/[subtopic]/[definition]/page.js

Example URLs:
  /jee
  /jee/physics
  /jee/physics/mechanics
  /jee/physics/mechanics/kinematics
  /jee/physics/mechanics/kinematics/motion
  /jee/physics/mechanics/kinematics/motion/velocity
  /jee/physics/mechanics/kinematics/motion/velocity/speed
